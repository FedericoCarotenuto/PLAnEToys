# PLAnEToys
Toy PLAnET - rainfall coupled model

This is a python implementation of the MATLAB-based [PLAnET model] (https://github.com/FedericoCarotenuto/PLAnET). This implementation has been coupled with a toy model that calculates the rainfall impacts of PLAnET microbial emissions (see [the ATBD] (ATBD_COMPLETE.md) for a comprehensive explanation of the algorithms). The whole thing is thought to be used interactively through a jupyter-lab interface in order to understand the impact of different parameters on emission and emission impact. 

# WARNING - READ THIS CAREFULLY!

While the PLAnET model itself has received scientific peer review (see ...), the remaining toy model has not. It is provided only for illustrative and teaching purposes and it is NOT supposed to be used for any real-world applications!
